# (Beautify)Clone of Sephora

# Technology used


- HTML
- CSS
- JAVA SCRIPT
- Node.js
- Mongo







## Features

- Login and Sign up 
- Search engine to search Products
- Sorting of Products (Price)
- Filter Products with respect to Category
- Purchase the Product
- Do payment
- Admin site 
- Add New Products
- Delete Data on the Website
